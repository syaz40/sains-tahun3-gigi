🦷 Misi Pakar Gigi: Projek PdPC Sains Tahun 3

Laman web interaktif ini dibina khas untuk membantu murid Tahun 3 (Kelas 3 UPM) mempelajari tentang jenis gigi manusia, fungsinya, dan contoh makanan yang sesuai melalui aktiviti digital yang menyeronokkan.


📋 Mengenai ProjekProjek ini merupakan alat bantu mengajar (PdPC) berasaskan web yang menggabungkan elemen permainan (gamification). Ia direka bentuk untuk dipaparkan pada skrin besar seperti Google TV 55 inci melalui casting dari iPad Pro.

Topik Pembelajaran:
Jenis Gigi: Gigi Kacip, Gigi Taring, Gigi Geraham.
Fungsi Gigi: Memotong, Mengoyak, Melumat.Contoh Makanan: Padanan makanan mengikut kesesuaian fungsi gigi.


🚀 Ciri-ciri Utama
Name Picker Beranimasi: Memilih murid secara rawak daripada senarai 37 orang murid 3 UPM untuk menjawab di depan kelas.

Sistem Drag & Drop: Aktiviti interaktif memadankan fungsi dan makanan ke slot gigi yang betul.

Library Makanan Rawak: Item makanan akan berubah secara rawak setiap kali murid baru dipilih untuk mengelakkan murid sekadar meniru jawapan rakan sebelumnya.

Timer & Skor: Memberikan cabaran dan elemen debaran kepada murid.

Paparan Optimasi TV: Teks gergasi dan kontras warna tinggi untuk memastikan murid di barisan belakang kelas dapat melihat dengan jelas.

🛠️ Teknologi Yang Digunakan
HTML5 & JavaScript (ES6+)Tailwind CSS (Untuk reka bentuk responsif)Google Fonts (Fredoka One & Quicksand)

🌐 Cara Pemasangan (GitHub Pages)
Pastikan fail index.html telah dimuat naik ke dalam repository anda.Pergi ke Settings > Pages.Pilih branch main dan folder / (root).Klik Save.Tunggu 1 minit dan laman web anda akan sedia untuk diakses melalui pautan yang diberikan oleh GitHub.

🎮 Cara Bermain
Buka pautan web pada iPad Pro.
Cast skrin iPad ke Google TV 55".
Klik butang "PUSING NAMA!" untuk memilih murid.
Murid yang terpilih akan datang ke depan dan menarik (drag) fungsi serta makanan ke kotak gigi yang betul.
Klik "MURID SETERUSNYA" untuk mengulang proses.
